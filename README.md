# HelloWorld

Este es mi primer proyecto en Java. Lo hice en Eclipse como parte de la clase.

## Qué hace el programa
Muestra un mensaje en pantalla y verifica si una persona es mayor de edad según la edad que está definida en el código.

## Archivos del proyecto
- HelloWorld.java: contiene el método main y toda la lógica del programa.
- IO.java: una clase que cree para poder imprimir mensajes de manera más fácil.

## Por qué cree el archivo IO.java
Al principio intenté usar IO.println() directamente en mi código, pero Java me marcaba error porque esa clase no existe por defecto. Investigué y me di cuenta de que debía crearla yo mismo para poder usarla. Por eso cree el archivo IO.java con varios métodos println que me permiten imprimir diferentes tipos de datos (texto, números, booleanos) de forma más sencilla.

## Cómo ejecutarlo
1. Abrir el proyecto en Eclipse.
2. Buscar la clase HelloWorld.java.
3. Hacer clic derecho y seleccionar Run As > Java Application.
